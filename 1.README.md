# Bright-TV-Case-Study
TV veiwership and subscription analysis with insightful report to the CEO on upscaling viewers subscription.

Project Overview.
BrightTV ‘s CEO has an objective to grow the company’s subscription base for this financial
year.I am tasked to provide insights that would assist CVM (Customer Value
Management) team in meeting this year’s objective.
The dataset attached contains information on the user profiles and viewer transactions for
BrightTV.

Problem Statement
While acquiring new viewers and returning the existing viewer subscription,quantitative analysis is needed .Introducing new 
programmes without profiling our existing viewers can results in loosing them.Buying new programmes without Data can be a lose
for Bright TV.

Aim
Discover actionable insights by looking at viewers trends on watching TV to lessen time spent on searching for relatable programmes.
While watching watching your favorite TV show then a pop-up message recommend what to watch.

Objectives
To have quantitative analysis on Exploratory Data  Analysis(EDA).
Look at trends and extract time spend on TV and what time of the day.
Deploy Dashborad for Continues analysis.
