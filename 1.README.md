# 📺 BrightTV Viewership Analytics Case Study

## 📌 Overview

* This project focuses on analyzing **BrightTV viewer transaction and user profile data** to generate business insights for executive decision-making.
* The primary objective is to support the **Customer Value Management (CVM)** team in growing the subscription base and improving customer engagement.
* The project combines:

  * Data analytics
  * SQL data processing
  * Dashboard development
  * Business intelligence reporting
  * AI-assisted insights
  * Executive storytelling
* Insights are presented through dashboards, charts, and strategic recommendations. 

---

# ❗ Problem Statement

* BrightTV aims to increase its subscription base and improve content consumption.
* The company requires deeper understanding of:

  * User behavior trends
  * Viewer consumption patterns
  * Factors influencing watch time
  * Low-consumption periods
  * Customer growth opportunities
* The CVM team needs actionable insights to drive user engagement and subscription growth. 

---

# 🎯 Project Objectives

* Analyze user profiles and viewer transaction data
* Identify user and usage trends
* Understand factors influencing content consumption
* Recommend content strategies for low-consumption periods
* Develop initiatives to grow BrightTV’s subscriber base
* Create dashboards and executive presentations
* Convert raw data into actionable business insights
